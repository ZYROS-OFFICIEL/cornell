Méthode Cornell de prise de notes

Ce site web a été fait par RAM

Comment utiliser la méthode Cornell :

1. Titre : Écrivez le sujet de vos notes en haut.

2. Points clés (colonne de gauche) : Notez les mots-clés, questions et concepts principaux.

3. Notes principales (colonne de droite) : Prenez vos notes détaillées pendant le cours ou la lecture.

4. Résumé (bas de page) : Après avoir terminé, écrivez un bref résumé des points essentiels.
